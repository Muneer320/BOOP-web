## #1 — CSS Variable Migration + Paper Texture

| Criteria | Rating |
|---|---|
| Visual Impact | ★★★★★ |
| Judging Impact | ★★★★ |
| User Impact | ★★★ |
| Dev Effort | 2h |
| **ROI Score** | **92/100** |

**Change:** Replace all CSS color/font/shadow variables in `index.css` with refined palette. Add paper texture background. Switch to Inter + Playfair Display + Source Serif 4.

**Before:** Muddy greens, serif-only, Trebuchet MS for UI, no texture, cold shadows
**After:** Intentional warm palette, elegant serif pairing, modern UI font, paper texture, warm ink shadows

**Screenshot value:** Side-by-side of any page before/after — instantly shows transformation
**Demo value:** Every page benefits simultaneously
**Blog value:** "We rebuilt our design system in 2 hours"
